Pro/Super/Twin (PST) Rules

Home | General Rules

  1. NQS Pro Stock at 1050# with 13" hitch, 39 C.I. V-Twin at 1050# with 13" hitch and NQS Pro Stock with open carburetor (no fuel injection) at 1025# with a 12.5" hitch.
  2. Pullers can pull their single cylinder tractors at each event either as Pro Stock or Pro Stock with open carburetor at any time. As an example qualify with small carb and pull finals with big carb.
  3. 48 cubic inch stock block K-series Kohler engines will be allowed to run with the same weight/hitch as the Pro Stock. 48c.i. engines will have the same rules as the 50.5 Pro stocks, with the following exceptions.Stock Block K-Series Kohler Engine, open carburetion (no fuel injection), no external welding allowed on the blocks (all-thread okay), if engine is sleeved, sleeve can not be externally visible. Must have head/block restraint.
  4. These rules can be changed in order to promote fairness of the type's tractors at anytime during the season.
